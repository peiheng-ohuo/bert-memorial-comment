# bert-memorial-comment

中文 | [English](README_EN.md)

这是我的一篇工作论文的代码

我尝试利用清代的奏折和朱批数据来研究皇帝的决策模式和专制权力。简单来说，我发现皇帝亲自做出指示的比例相当低，大部分事务的决策权都被交给了官僚系统。而且这种决策权的转交非常程序化，这是我希望用深度学习来证明的。

我对奏折题目和特定朱批标签之间的关系训练了微调[GuwenBERT](https://github.com/Ethan-yt/guwenbert)模型，利用模型较好的学习结果来说明皇帝转交决策权过程的程序化。

我会陆续将代码和数据上传到这里。代码已经上传，手工分类的朱批标签数据在论文发表后也会上传。
